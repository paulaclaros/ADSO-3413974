# 🌱 Instrumentos de Recolección de Información Aplicados al Proyecto AgroWeb Huila
## 📌 Proyecto
**AgroWeb Huila**  
Sistema web para la gestión y monitoreo de cultivos de café.
Antes de desarrollar la página web, fue necesario **recolectar información directamente de los agricultores y del proceso del cultivo**.
Esto permite conocer:
- cómo trabajan actualmente
- qué problemas tienen
- qué datos necesitan registrar
- qué funciones debe tener el sistema
---
# 🧩 Aplicación de los Instrumentos de Recolección de Información en el Proyecto
Los instrumentos de recolección de información permiten obtener datos reales del entorno para diseñar una solución tecnológica adecuada.
En este proyecto, se aplican de la siguiente manera:
---
## 1. Entrevista
La entrevista consiste en hablar directamente con los agricultores para conocer cómo administran actualmente sus cultivos.
### 📌 Aplicación en el proyecto
Se realizan preguntas como:
- ¿Cómo registran el riego?
- ¿Cada cuánto fertilizan?
- ¿Dónde guardan la información?
- ¿Qué problemas tienen al llevar el control manual?
### 🎯 Relación con el proyecto
Gracias a este instrumento se identificó que muchos agricultores usan:
- cuadernos
- hojas sueltas
- la memoria
Esto genera pérdida de información y errores en la toma de decisiones.
---
## 2. Encuesta
La encuesta permite obtener información de varios agricultores de forma rápida.
### 📌 Aplicación en el proyecto
Ejemplo de preguntas:
- ¿Cuántas veces riegan por semana?
- ¿Cuántas plantas tienen?
- ¿Usan celular o computador?
- ¿Saben utilizar plataformas web?
### 🎯 Relación con el proyecto
Permite conocer el perfil del usuario final para diseñar una plataforma:
- fácil de usar
- responsive
- accesible desde dispositivos móviles
---
## 3. Observación
La observación consiste en analizar directamente cómo se realiza el proceso agrícola.
### 📌 Aplicación en el proyecto
Se observa cómo el agricultor:
- revisa las plantas
- aplica agua
- fertiliza
- registra información
- hace seguimiento al crecimiento
### 🎯 Relación con el proyecto
Permite comprender el flujo real del proceso.
```text
Revisar cultivo → medir crecimiento → regar → registrar fecha
```
